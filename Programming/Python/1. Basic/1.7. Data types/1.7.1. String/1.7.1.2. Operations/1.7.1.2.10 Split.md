


  
**.split()** - allows to split given string into separate letters and turn it into a list. Split by default divides on whitespaces, but it uses a parameter which can change that.  
  

```python
new = 'hwewlwlwo'  
new2 = new.split('w')  
print(new2)  
  
------------------------------  
['h', 'e', 'l', 'l', 'o']
```
