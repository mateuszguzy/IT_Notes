


  
Having a dictionary with prepared data we can use simple function **pandas.DataFrame** to generate DataFrame  
  

```python
data\_dict = {  
'students': ['Amy', 'Pete'],  
'grades': [5, 4]  
}  
data = pandas.DataFrame(data\_dict)  
------------------------------  
  
  students  grades  
0      Amy       5  
1     Pete       4
```
